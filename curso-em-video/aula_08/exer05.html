Operadores Relacionais:

  • > : Maior que...
  • < : Menor que...
  • >= : Maior ou igual a...
  • <= : Menor ou igual a...
  • == : Igual a...
  • != : É diferente de...

Exemplos:

  1. Preço >= 200.00; //O preço é maior ou igual a 200.00?
  2. Idade < 18; //A idade é menor que 18?
  3. Curso == "Javascript"; // O curso é Javascript?
  4. n1 != n2; // O n1 é diferente de n2?

Operador de Identidade:

  • '===' -> Operador de igualdade restrita.
  • 5 == 5; -> True (int - int)
  • 5 == '5'; -> True (int - str)
  • 5 === '5'; -> False (int - str) - Esse operador '===', além de comparar o tipo, ele também compara se o valor é igual, ou seja, o 5 é do mesmo tipo primitivo e tem o mesmo valor que o '5'?
  • 5 === 5; -> True (int - int) - Agora sim, ambos tem o mesmo valor e são do mesmo tipo.
  • 5 !== '5'; -> True (int - str) - São de tipos diferentes mas ambos tem o mesmo valor.

    • Operadores Lógicos:

  • ! : Negação
  • && : Conjução ou 'and'
  • || : Disjunção ou 'or'

    • Exemplos:

    Negação

  • !True = False
  • !False = True

    • Conjução ou 'and'

  • True && True = True
  • True && False = False
  • False && True = False
  • False && False = False

    • Disjunção ou 'or'

  • True || True = True
  • True || False = True
  • False || True = True
  • False || False = False
