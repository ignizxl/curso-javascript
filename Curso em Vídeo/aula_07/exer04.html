Operadores Aritméticos:

  • + = Adição
  • - = Subtração
  • * = Multiplicação
  • / = Divisão
  • % = Resto da Divisão
  • ** = Potenciação

Ordem de Precedência:

  1. ()
  2. **
  3. *, /, %
  4. +, -

Auto-Atribuições:

var x = 5;

  • x+= 3; -> É o mesmo que: x = x + 3;
  • x-= 4; -> É o mesmo que: x = x - 4;
  • x*= 5; -> É o mesmo que: x = x * 5;
  • x/= 6; -> É o mesmo que: x = x / 6;
  • x%= 2; -> É o mesmo que: x = x % 2;
  • x**= 3; -> É o mesmo que: x = x ** 3;

    • Incremento e Decremento:

    var x = 5;

  • x++; -> É o mesmo que: x = x + 1;
  • x--; -> É o mesmo que: x = x - 1;
