Tipos Primitos:

  1. Number
    Exemplos:
    int : 10,
    float : 5.0,
    double : 5.0.
  2. String
    Exemplo: "Olá, Mundo!"
  3. Boolean
    Exemplo: True or False.
  4. Null
    Descrição: O Null é um object nulo e sem valor.
    5. Array
    Descrição: O Array é uma variável composta que é capaz de armazenar vários valores.
    Exemplo: Alimentos = ['Arroz', 'Macarrão', 'Feijão', 'Carne']
  5. Object
    Descrição: Um objeto é uma coleção de propriedades, e uma propriedade
    é uma associação entre um nome (ou chave) e um valor.
    Exemplo: let pessoa = {
    nome = 'João',
    idade = 18
    }

    'nome' representa a 'chave' e 'João' representa o 'valor'.
  6. Undefined
    Descrição: Uma variável que não teve um valor atribuído.
  7. Esses são apenas alguns Tipos Primitos...
